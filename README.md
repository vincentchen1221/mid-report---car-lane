# 期中報告
## 動機與目的
車道線偵測是現代智慧車輛技術的基石，其最核心的應用動機，是提升駕駛的「安全性」與「舒適性」，並最終實現「自動駕駛」

## 優化
### 車道線平均
蒐集所有偵測到的線段，過濾後取「斜率」和「截距」的平均值，產生一條平滑、穩定的車道線，標示車道邊緣
### 影格記憶
使用全域變數記住上一幀的車道線，當此幀偵測失敗時，能沿用上一幀的結果，防止車道線閃爍或消失
